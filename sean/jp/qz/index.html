プライバシーポリシー

一、はじめに
当社が提供する投資情報サービスをご利用いただき、誠にありがとうございます。当社はユーザーの個人情報保護を重視しており、本プライバシーポリシーは、LINE で当社を追加し「7」を送信して投資情報を取得する際に、当社が個人情報を収集、利用、保存、保護する方法と、ユーザーの権利について明かすものです。本サービスをご利用いただく前に、本プライバシーポリシーの内容を十分にお読みいただき、ご理解いただけますようお願いいたします。
二、情報の収集
必要な情報の収集：LINE で当社を追加し「7」を送信して投資情報を取得する際、当社は LINE アカウント ID のみを収集いたします。この情報はサービスの利用者を確認し、個別に投資情報を配信するために使用いたします。その他、氏名、LINE アカウント以外の連絡先、身分証明番号、財務状況などの敏感な個人情報は、原則として収集いたしません。 不要な情報の収集（場合により）：サービスを利用する際、LINE でのチャット記録から、自発的に提供いただいた情報（例えば、投資に関する質問など）を収集する場合があります。ただし、これらの情報の提供はユーザーの自由な意思に基づくものであり、より質の高いサービスを提供するためのみに使用いたします。
三、情報の利用
収集した LINE アカウント ID などの情報は、約束した投資情報（明日の上昇見通しのある銘柄の紹介、週に 60％の利益を約束する無料の銘柄に関する情報、最新の株価上昇ニュースなど）を配信するためのみに使用いたします。 当社は、ユーザーの個人情報を第三者に売却、貸し出し、または商業的なマーケティング目的で共有することはありません。ただし、以下の場合には、個人情報を共有することがあります： 法律法規の要求に基づき、司法機関、行政機関などの権限ある機関の調査、執行業務に協力する場合； 合併、買収、破産清算などの事業または資産の譲渡に関連する場合で、法的に適切な手続きに従っている場合。
四、情報の保存と保護
当社は、適切な技術的および管理的措置を講じて、ユーザーの個人情報のセキュリティを保障し、情報漏えい、損壊、消失を防止いたします。収集した情報は暗号化して保存し、関係者の情報アクセス権限を制限し、サービスの提供に必要な従業員のみが情報にアクセスし、処理できるようにしています。 当社は、サービスの提供に必要な期間のみ個人情報を保存します。サービスの終了時またはユーザーから情報削除の要求があった場合、法律法規の要求に従って、個人情報を削除または匿名化処理いたします。
五、ユーザーの権利
閲覧権：いつでも LINE で当社に連絡することで、当社が収集した個人情報の内容を閲覧する権利があります。 訂正権：当社が収集した個人情報に誤りがあることを発見した場合は、LINE で訂正の請求をすることができます。当社は速やかに確認し、訂正いたします。 削除権：法律法規の定める条件に合致する場合、LINE で個人情報の削除を請求することができます。例えば、当社のサービスを利用しなくなった場合、要求に応じて関連情報を削除いたしますが、法律法規の規定や法定义務の履行などの場合は除きます。 同意撤回権：LINE でのやり取りを停止することで、当社が個人情報を収集・利用することに対する同意をいつでも撤回することができます。ただし、同意の撤回により、当社が提供する投資情報サービスの利用が制限される場合があります。
六、ポリシーの更新
当社は、法律法規の変更や事業発展のニーズに応じて、本プライバシーポリシーを更新する場合があります。更新後のプライバシーポリシーは、当社のサービス関連ページまたは LINE メッセージなどを通じて公表します。プライバシーポリシーが更新された後も、引き続き当社のサービスをご利用いただいた場合、更新後の内容を十分に読み、理解し、受け入れたものとみなします。
七、お問い合わせ
本プライバシーポリシーに関するご質問、ご意見、ご提案がある場合、または個人情報の権利を行使したい場合は、いつでも LINE で当社にご連絡ください。当社は適切な期間内に回答し、ご質問に対応いたします。
